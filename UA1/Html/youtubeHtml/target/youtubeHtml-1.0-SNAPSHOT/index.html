Reproduciendo un video de YouTube en HTML

Reproducción automática de YouTube

Valor 0 (predeterminado): el video no se reproducirá automáticamente cuando se cargue el reproductor.

Valor 1: el video se reproducirá automáticamente cuando se cargue el reproductor.

Bucle de YouTube

Valor 0 (predeterminado): el video se reproducirá solo una vez.

Valor 1: el video se repetirá (para siempre).

Controles de YouTube

Valor 0: los controles del reproductor no se muestran.

Valor 1 (predeterminado): la pantalla de controles del reproductor.
